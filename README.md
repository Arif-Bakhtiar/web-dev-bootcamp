# Goods Panda – E-commerce Landing Page

A modern and responsive e-commerce landing page built using **HTML5** and **CSS3**.  
The project includes a hero product showcase, category cards, featured products, and a newsletter subscription section.

## Features

- Responsive landing page layout
- Hero section with featured product
- Category cards (Watch, Bag, Shoes)
- Featured product sections (Shoes & Backpacks)
- Newsletter subscription form
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- Flexbox for layout
- Responsive design with media queries
